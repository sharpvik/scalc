# Scalc

**Scalc** is a stack-based calculator app that runs in a terminal. I built it using Python3.



## How to Run

```bash
git clone ...
cd scalc
chmod +x main.py
./main.py
# or just 'python3 main.py'
```



## How it Works

| Operator | Explanation       |
|:--------:|:------------------|
| +        | Addition          |
| -        | Subtraction       |
| *        | Multiplication    |
| /        | Division          |
| ^        | Power             |
| !        | Factorial         |
| =        | Peek              |
| #        | Pop               |
| $        | Emit (Peek + Pop) |
| ?        | Show the Stack    |
| q        | Quit              |



## Dependencies

* Python 3.7
